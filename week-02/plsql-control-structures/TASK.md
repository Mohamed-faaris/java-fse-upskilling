# Exercise 1: Control Structures

**Source:** PL SQL programming/PLSQL_Exercises.docx

## Goal
Write PL/SQL blocks that use control structures to apply discounts, set VIP flags, and send reminders.

## Scenarios
1. Apply 1% discount to loan interest rates for customers above 60 years old
2. Set IsVIP flag to TRUE for customers with balance over $10,000
3. Send reminders to customers whose loans are due within the next 30 days

## Tasks
- Write PL/SQL blocks that loop through customers and apply business rules
- Implement conditional logic for age-based discount and balance-based VIP status
- Fetch loans due in 30 days and print reminder messages
